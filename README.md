# 🎮 Rock Paper Scissors Game

A rock-paper-scissors game built with Python using **Tkinter** for the graphical user interface and **Pygame** for immersive sound effects and background music. It features an integrated score tracker and interactive in-game challenges.

---

## ✨ Features

- **Interactive UI:** A modern and user-friendly desktop application built entirely with Tkinter.
- **Sound Effects & Music:** Dynamic background music and arcade-style sound effects powered by Pygame Mixer.
- **Score System:** Keeps track of your score, the computer's score, and draws in real-time.
- **In-Game Challenges:** Goal-oriented achievements (e.g., "Achieve 10 wins!") to keep the gameplay engaging.
- **AFK Protection:** Built-in timeout warning if the user fails to make a choice within 2 seconds.

---

## 🚀 How to Play (For Casual Players)

If you just want to play the game without setting up Python or any dependencies:
1. Navigate to the **Releases** section on the right side of this GitHub repository page.
2. Download the packaged executable (`main.exe`) along with the `assests` folder.
3. Make sure the executable file and the `assests` folder remain in the same directory.
4. Double-click `main.exe` and enjoy the game!

## 💻 Developer Setup (Running from Source)

If you want to view, modify, or run the source code on your local machine, follow these steps:

### Prerequisites
Make sure you have **Python 3.x** installed on your system. 

### Installation

1. Clone this repository to your local machine.

2. Install the required external library (Pygame):
pip install -r requirements.txt

3. Launch the game by executing the main script:
python main.py
